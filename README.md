This project is an intelligent, secure, and college-exclusive web platform designed to facilitate the buying and selling of second-hand books and gadgets among college students. The core innovation lies in the use of Artificial Intelligence to streamline the product listing process, making selling easier and faster.

Type: Web Application

Level: Final Year / Major Mini Project

Domain: Web Development, Artificial Intelligence

Target Users: College Students

Problem Solved: Providing a secure, trusted, and intelligent platform for students to exchange used goods, overcoming the friction of manual product listing.

-------------------------------------------------------------------------------------------
| Role             | Capabilities                                                          |
| ---------------- | --------------------------------------------------------------------- |
| Student (Seller) | Login/Register, Sell products (AI-assisted listing), Chat with buyers |
| Student (Buyer)  | Login/Register, Browse/Search products, Chat with sellers             |
| Admin            | Moderate listings, Manage user accounts, Ensure platform security     |
-------------------------------------------------------------------------------------------|
